  lxc config set core.proxy_http http://16.85.88.10:8080
  lxc config set core.proxy_https http://16.85.88.10:8080

- create various loopback devices to be used as secondary device for storage nodes
- create lxc network for data, external

  - run 'ansible-user' playbook
